\doxysubsection{Detailed Description}
The abstract base class for all entries in a \doxylink{class_q_c_p_legend}{QCPLegend}. 

It defines a very basic interface for entries in a \doxylink{class_q_c_p_legend}{QCPLegend}. For representing plottables in the legend, the subclass \doxylink{class_q_c_p_plottable_legend_item}{QCPPlottable\+Legend\+Item} is more suitable.

Only derive directly from this class when you need absolute freedom (e.\+g. a custom legend entry that\textquotesingle{}s not even associated with a plottable).
