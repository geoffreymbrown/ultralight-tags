\doxysubsection{Detailed Description}
The abstract base class for layouts. 

This is an abstract base class for layout elements whose main purpose is to define the position and size of other child layout elements. In most cases, layouts don\textquotesingle{}t draw anything themselves (but there are exceptions to this, e.\+g. \doxylink{class_q_c_p_legend}{QCPLegend}).

\doxylink{class_q_c_p_layout}{QCPLayout} derives from \doxylink{class_q_c_p_layout_element}{QCPLayout\+Element}, and thus can itself be nested in other layouts.

\doxylink{class_q_c_p_layout}{QCPLayout} introduces a common interface for accessing and manipulating the child elements. Those functions are most notably \doxylink{class_q_c_p_layout_acc6012e7dacc55c4282942c31c45b8d9}{element\+Count}, \doxylink{class_q_c_p_layout_ab22bd9153d855a35ab4010267ae36a69}{element\+At}, \doxylink{class_q_c_p_layout_aae3fc1bd5656aa737f18f0779789fd79}{take\+At}, \doxylink{class_q_c_p_layout_ad110be0de976cd8a817513a4a53d53db}{take}, \doxylink{class_q_c_p_layout_a41e6ac049143866e8f8b4964efab01b2}{simplify}, \doxylink{class_q_c_p_layout_a2403f684fee3ce47132faaeed00bb066}{remove\+At}, \doxylink{class_q_c_p_layout_a6c58f537d8086f352576ab7c5b15d0bc}{remove} and \doxylink{class_q_c_p_layout_a02883bdf2769b5b227f0232dba1ac4ee}{clear}. Individual subclasses may add more functions to this interface which are more specialized to the form of the layout. For example, \doxylink{class_q_c_p_layout_grid}{QCPLayout\+Grid} adds functions that take row and column indices to access cells of the layout grid more conveniently.

Since this is an abstract base class, you can\textquotesingle{}t instantiate it directly. Rather use one of its subclasses like \doxylink{class_q_c_p_layout_grid}{QCPLayout\+Grid} or \doxylink{class_q_c_p_layout_inset}{QCPLayout\+Inset}.

For a general introduction to the layout system, see the dedicated documentation page The Layout System.
