\doxysubsection{Detailed Description}
Provides rect/rubber-\/band data selection and range zoom interaction. 

\doxylink{class_q_c_p_selection_rect}{QCPSelection\+Rect} is used by \doxylink{class_q_custom_plot}{QCustom\+Plot} when the \doxylink{class_q_custom_plot_a810ef958ebe84db661c7288b526c0deb}{QCustom\+Plot\+::set\+Selection\+Rect\+Mode} is not \doxylink{namespace_q_c_p_ac9aa4d6d81ac76b094f9af9ad2d3aacfa894e973fdf7efc6174a9de50babb19fd}{QCP\+::srm\+None}. When the user drags the mouse across the plot, the current selection rect instance (\doxylink{class_q_custom_plot_a0c09f96df15faa4799ad7051bb16cf33}{QCustom\+Plot\+::set\+Selection\+Rect}) is forwarded these events and makes sure an according rect shape is drawn. At the begin, during, and after completion of the interaction, it emits the corresponding signals \doxylink{class_q_c_p_selection_rect_a7b7162d19f4f2174d3644ff1a5d335aa}{started}, \doxylink{class_q_c_p_selection_rect_a1bab11026bca52740c2e6682623e6964}{changed}, \doxylink{class_q_c_p_selection_rect_aeb82009393c90130102dccf36477b906}{canceled}, and \doxylink{class_q_c_p_selection_rect_a15a43542e1f7b953a44c260b419e6d2c}{accepted}.

The \doxylink{class_q_custom_plot}{QCustom\+Plot} instance connects own slots to the current selection rect instance, in order to react to an accepted selection rect interaction accordingly.

\doxylink{class_q_c_p_selection_rect_ad27c1569c6ea8fa48e24b81e2a302df3}{is\+Active} can be used to check whether the selection rect is currently active. An ongoing selection interaction can be cancelled programmatically via calling \doxylink{class_q_c_p_selection_rect_af67bc58f4f5ce9a4dc420b9c42de235a}{cancel} at any time.

The appearance of the selection rect can be controlled via \doxylink{class_q_c_p_selection_rect_ada20b7fb1b2dcbe50523262636b06963}{set\+Pen} and \doxylink{class_q_c_p_selection_rect_ab0c66f1484418782efa01f4153611080}{set\+Brush}.

If you wish to provide custom behaviour, e.\+g. a different visual representation of the selection rect (QCPSelection\+Rect\+::draw), you can subclass \doxylink{class_q_c_p_selection_rect}{QCPSelection\+Rect} and pass an instance of your subclass to \doxylink{class_q_custom_plot_a0c09f96df15faa4799ad7051bb16cf33}{QCustom\+Plot\+::set\+Selection\+Rect}.
