\doxysubsection{Detailed Description}
A paint buffer based on QPixmap, using software raster rendering. 

This paint buffer is the default and fall-\/back paint buffer which uses software rendering and QPixmap as internal buffer. It is used if \doxylink{class_q_custom_plot_a7db1adc09016329f3aef7c60da935789}{QCustom\+Plot\+::set\+Open\+Gl} is false.
